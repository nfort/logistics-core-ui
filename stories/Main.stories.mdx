# Logistics-core-ui

### Установка

```bash
npm install @nfort/logistics-core-ui @holism/components@0.9.9
```

Зависимости

### Formik-ui
Содержит компоненты, в которых были исправлены ошибки в @holism/components

```bash
npm install @nfort/formik-ui
```

### Formik-holism
Обертка контролов для Formik

```bash
npm install @nfort/formik-holism
```
